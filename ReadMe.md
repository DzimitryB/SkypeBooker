# Skype Bot на Python

Этот бот отправляет сообщения в указанную группу Skype.

## Установка

1. Клонируйте репозиторий или скачайте файлы.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Пропишите ваши учетные данные в ".env" файле.

## Запуск
``` bash
   python main.py
